1. Website Tidak Menampilkan Data

Penyebab: URL Google Script salah atau belum di-deploy sebagai Web App.

Solusi:
- Buka Google Apps Script dan pilih Deploy → New Deployment.
- Pastikan akses "Anyone" atau "Anyone with link".
- Ganti URL lama di file PHP atau JavaScript dengan yang baru.

2. ESP8266 Tidak Bisa Kirim Data

Penyebab: API Google Script tidak merespons, atau format HTTPClient salah.

Solusi:
- Gunakan format baru: http.begin(WiFiClient, scriptURL);
- Pastikan WiFi terhubung (uji dengan WiFi.status()).
- Tes URL Google Script dari browser, pastikan bisa dibuka.

3. Data Tidak Masuk ke Google Sheets

Penyebab: Spreadsheet ID salah atau variabel POST tidak cocok.

Solusi:
- Periksa apakah Apps Script memakai doPost(e) dengan parameter sesuai JSON.
- Pastikan kolom di spreadsheet sesuai urutan data kiriman.
- Tambahkan Logger.log(e.postData.contents) untuk debug.

4. Website Error atau Blank

Penyebab: Kesalahan sintaks PHP/JS atau file tidak ditemukan.

Solusi:
- Buka Console (Ctrl+Shift+I) untuk melihat pesan error.
- Pastikan semua file (sidebar.php, script.js, dll) ada di folder yang benar.
- Jika menggunakan server lokal, jalankan dengan localhost atau XAMPP.
